# 🚀 Orbital — E-commerce de Astronomia

Bem-vindo ao **Orbital**, um e-commerce 100% frontend focado em levar o universo até a casa dos clientes. De telescópios profissionais a luminárias de planetas hiper-realistas, este projeto foi desenvolvido para oferecer uma experiência de navegação fluida, visualmente imersiva e moderna.

---

## 🌌 Escopo do Projeto

O objetivo principal deste projeto foi construir uma SPA (Single Page Application) focada na experiência do usuário (UX/UI), cobrindo o fluxo essencial de um e-commerce sem a necessidade de um ecossistema de banco de dados robusto.

### Principais Funcionalidades:
*   **Catálogo Dinâmico:** Navegação e filtragem de produtos por categorias (Camisetas, Telescópios, Decorações, Acessórios).
*   **Página Interna de Produto:** Visualização detalhada de especificações, preço e imagens do item selecionado.
*   **Carrinho Lateral (Sidebar Drawer):** Um carrinho responsivo que se sobrepõe à tela atual pelo lado direito, permitindo gerenciar itens sem perder o contexto da navegação.
*   **Cálculo de Frete Dinâmico:** Integração com API de busca de CEP para simulação do valor de entrega direto no carrinho ou na página do produto.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as ferramentas mais modernas do ecossistema Frontend:

*   **[React](https://reactjs.org/):** Para a criação de componentes modulares e gerenciamento de estado (Context API para o carrinho).
*   **[Tailwind CSS](https://tailwindcss.com/):** Para a estilização rápida, responsiva e com um visual *dark space* elegante.
*   **[Vite](https://vitejs.dev/):** Ferramenta de build ultra-rápida para o ambiente de desenvolvimento.
*   **JSON Local:** Simulação de banco de dados através de um arquivo estático para popular o catálogo.
